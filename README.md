# Ash Phoenix / Filter Form Example

An example using [AshPhoenix.FilterForm](https://ash-hq.org/docs/module/ash_phoenix/latest/ashphoenix-filterform), to create nested forms that generate boolean expressions to be used in filters. See `MyAppWeb.FilterFormLive`.

## Install

Clone and run `mix setup` as normal - note that the database will be seeded with some sample data.
